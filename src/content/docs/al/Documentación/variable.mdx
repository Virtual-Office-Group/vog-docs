Entre las funciones variables se incluyen:

- Clear
- ClearAll

Al trabajar con variables, cada una de ellas con un tipo de datos diferentes, si quiere asignar un valor de un tipo de datos a otro, puede usar las conversiones de tipo automáticas. Si esta opción no es posible, use las siguientes funciones:

- Evaluate
- Format

## Funciones Clear y ClearAll

La función **Clear** le ayuda a despejar o restablecer una variable. Esta función reiniciará el valor predeterminado de la variable.

```jsx
var
   myText: Text[20];

myText := UserId;
Clear(myText);      // Sets the variable myText to an empty string.
```

La función **ClearAll** restablece todas las variables (excepto la variable Rec) a sus valores predeterminados. También borra todas las claves y filtros que se han configurado.

## Función Evaluate

Puede usar la función **Evaluate** para convertir una variable del tipo de datos texto (código o texto) a otro tipo de datos (que no sea de texto)

```jsx
var
   myText: Text[20];
   myInteger: Integer;

myText := '5';
Evaluate(myInteger, myText);
```

## Función Format

La función **Format** convierte un tipo de datos a tipo de datos de texto.

```jsx
var
   myText: Text[20];
   myInteger: Integer;

myInteger := 3;
myText := Format(myInteger);
```
