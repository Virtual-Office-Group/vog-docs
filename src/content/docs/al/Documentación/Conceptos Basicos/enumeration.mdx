El tipo de datos **Option** es un tipo de enumerador de base cero y, por lo tanto, permite convertir valores de opción en enteros. Por esa razón, el tipo de datos **Option** es un tipo de datos numéricos, ya que los valores de opción se almacenan como números enteros.

Para definir una variable de tipo **Option**, no puede usar la propiedad **OptionMembers** que se usa en un campo de tipo de datos **Option**. Debe enumerar las opciones disponibles como una lista separada por comas después de definir la variable.

![using-option-data-type-al-ss.png](./../../../../../assets/documentation/basic/using-option-data-type-al-ss.webp)

Si quiere reutilizar el mismo tipo **Option** en otros objetos (como otras unidades de código, páginas o tablas), debe redefinir todos los valores disponibles. Más adelante, si decide agregar un valor adicional, debe modificar todos los objetos con dicho valor adicional. Las opciones de una tabla no se pueden extender con una extensión de tabla.
