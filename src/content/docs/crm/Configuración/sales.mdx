# Configuración CRM Sales Virtual Office Group.

Para la implementación del nuevo entorno de Dynamics 365 Sales, se lleva a cabo la configuración y personalización de las entidades involucradas, junto con las respectivas pruebas de funcionamiento.


## Configuración de Ventas: Unidad de Negocio

***Concepto:***

En Dynamics 365, una unidad de negocio representa una agrupación lógica de actividades comerciales relacionadas, que permite organizar y segmentar operaciones según la estructura organizacional.

### Proceso de configuración

1. *Acceda a **Configuración avanzada**.*
2. *Diríjase a **Seguridad**.*
3. *Seleccione **Unidad de negocio** para definir y gestionar las unidades requeridas.*

Este enfoque facilita la administración de roles, permisos y datos de manera alineada con la estructura operativa de la organización.

![Unidad de Negocio](../../../../assets/crm/configuration/sales/1.png)

Una vez configurada la unidad de negocio, se procede a asignar los usuarios correspondientes para definir los permisos y alcances de acceso dentro de la estructura organizacional.

![Asignacion de usuarios](../../../../assets/crm/configuration/sales/2.png)
